### For loop
```cs
for (statement 1; statement 2; statement 3) 
{
  // code block to be executed
}
```
### foreach Loop
```cs
foreach (type variableName in arrayName) 
{
  // code block to be executed
}
```

### Switch
```cs
switch(expression) 
{
  case x:
    // code block
    break;
  case y:
    // code block
    break;
  default:
    // code block
    break;
}
```
